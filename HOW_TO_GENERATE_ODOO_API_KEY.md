# How to Generate Odoo API Key

## Step 1: Access Account Security
1. In Odoo, click on your **profile picture** or **name** (top right corner)
2. Click **Preferences** or **My Profile**
3. Click on **Account Security** tab

## Step 2: Generate New API Key
1. Scroll to **API Keys** section
2. Click **New API Key** button
3. Enter a description: "Razorpay Webhook Integration"
4. Click **Generate Key**

## Step 3: Copy the API Key
**IMPORTANT:** The API key will be shown **only once**!
- Copy the entire key (it's usually a long string)
- Save it temporarily in a secure place (you'll paste it in Replit next)

## Step 4: Provide to Replit (Next Step)
After you generate the key, I'll ask you to paste it into Replit's **Secrets** tab.

**Never share API keys in chat or publicly!**
